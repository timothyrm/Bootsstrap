GUIDE FÖR BOOTRSTRAP 4

Element

Ett element är en något Default inom html, det är den som har en start tag och en slut tag, så som p och div.

HTML

HTML står för HyperText Markup Language och används till att skapa hemsidor, här är ett exempel:

ID

ID används för att kunna rikta en specefik style deklaration i en stylesheet, den är ganska lik class, bara att skillnaden är att ett id ska vara unikt på en webbsida. Det finns ingen speciell regler för vilka element som får använda sig av id, men typiskt använder man id på större områden som exempelvis sidomenyer, behållare för artikeltext, sidhuvud, sidfot och så vidare.

Class

Class fungerar på ungefär exakt samma sätt som id men med dessa skillnader: 1. Samma class kan användas till flera element på en webbsida. 2. Du kan inte direktlänka till en class.

Meta

Meta taggen definierar metadata om HTML-dokumentet. Den är data information om data. meta taggen visas alltid i elementet head och används vanligtvis för att ange teckenuppsättning, sidbeskrivning, nyckelord, dokumentförfattare och visningsinställninga Metadata visas inte på sidan, men dem analyseras av maskinen. Webbläsare, sökmotorer och andra webbtjänster använder metadata.

Responsive

Responsive web design är hur man får hemsidan att se snygg ut på alla olika enheter (mobil, dator osv). Responsive är hur man använder sig utav html och css för att omforma allt så det ser br a ut överallt.

http

http står i början av en hemsida, dens uppgift är att när man tex klickar på en länk så kommer du begära ankomst vilket du får när hemsidan kommer upp. Dock är http inte så säkert då dens fokus är att visa informationen som kommer mer än hur informationen kommer.

https

https är likadant som http bara att den är mer avancerad och mycket säkrare att använda. Om hemsidan begär att du ska skriva lödsenord så är det viktigt att kolla att det är https först.

compressed data

compressed data är hur man modifierar, konverterar små bitar av data så att det tar mindre utrymme.

CDN

CDN står för Content Distribution Network och är ett nätverk av noder på olika geografiska platser som hjälper till att distribuera innehåll

local bootstrap

Local bootstrap är när man skickar in en link istället för laddar ner boostrap. Det är det snabbaste sättet att komma igång med bootstrap.

JQuery

JQuery är en typ av kod, såsom javascript men den tar från javascript libraryn. Den förkortar allt också så det är som "write less, do more".

Flexbox

Flexbox är ett modernare sätt att skapa layout på webbsidor. Det är som det låter, man kan flytta runt boxar.

grid

CSS Grid Layout Module är ett grid-based layout system, med rader och kolumner, Detta gör det lättaree att designa hemsidor utan att använda sig utav float och positioning.

code snippets

En HTML snippet är en liten mängd av source code i en HTML markup.
